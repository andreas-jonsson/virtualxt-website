VirtualXT


VirtualXT is an IBM PC/XT emulator that runs on modern hardware and operating systems. It is designed to be simple and lightweight yet still capable enough to run a large library of old application and games.

Featured hardware:

  • * Intel 8088 or NEC V20 CPU
  • * 1MB RAM
  • * CGA/Hercules or MDA graphics card
  • * Super PC/Turbo XT BIOS 3.1 + VXTCGA
  • * Keyboard controller with 83-key XT-style keyboard
  • * Serial port with Microsoft 2-button mouse
  • * Floppy and hard disk controller
  • * PC speaker

VirtualXT is in early development and there is a lot of work to be done. You can expect crashes and incompatibilities.

Opensource

VirtualXT is completely free and released under the GPL3 license.

Written in Go

VirtualXT is written entirely in Go with an optional dependency to SDL2.

Lightweight

One of the projects top priorities is to make the code simple, easy to understand, maintain and improve.

Opensource BIOS

VirtualXT shipps with Super PC/Turbo XT BIOS 3.1 and it's own custom VXTCGA video BIOS.
